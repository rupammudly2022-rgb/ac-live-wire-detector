# AC Live Wire Detection Circuit

This project is a simple circuit used to detect AC live wire without direct contact.

The circuit uses a small antenna to capture electromagnetic field near live wire.
The signal is amplified using transistor stages and used to drive an LED indicator.

Components used:
Transistors
Resistors
Capacitors
LED
Antenna wire
Veroboard

Description:
When the antenna is brought near AC live wire,
electromagnetic field is detected and amplified,
which turns ON the LED.
